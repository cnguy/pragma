Welcome to Pragma!

Pragma is a self-hosted, open-source, personal note taking app. It's written in Rust and ReasonML.

Pragma runs in your browser and syncs all your notes back to the server transparently so you'll never lose your notes.

All your notes are also available offline in your browser, so always available to you.

Pragma works like this:

  • Notes are organized in notebooks.
  • Notebooks contain notes.
  • Notes consist of content blocks.
    • There are two types of content blocks:
      • Text (like this one!).
      • Code (with syntax highlighting!).
    • Currently, notes can only have one content block but soon you'll be able to create multiple blocks per note.

Please refer to the docs or open an issue if you have any questions!
